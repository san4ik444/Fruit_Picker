Актуальное распознавание яблок в реальном времени с использованием нейросети YOLOv5. Этот репозиторий — программное сердце проекта «Fruit Picker»: автономного роботизированного комплекса на базе коптера «Геоскан Пионер» для точного и эффективного сбора урожая.

🎯 Назначение репозитория
Здесь собраны все цифровые артефакты проекта:

Код компьютерного зрения — для обнаружения яблок на видео с камеры дрона.

3D-модели — детали вакуумного захвата и защиты коптера.

Техническая документация — инструкции по сборке и настройке.

📂 Структура репозитория

🧠 Как это работает (Программная часть)
Нейросеть YOLOv5 обрабатывает каждый кадр с камеры.

Алгоритм ищет объекты, похожие на яблоки, по цвету, форме и текстуре.

При обнаружении вокруг яблока рисуется bounding box (рамка) с меткой apple и процентом уверенности.

Координаты яблока передаются в систему управления дроном для наведения захвата.

💡 Особенность: Модель можно дообучить на собственных изображениях яблок (например, для конкретных сортов в Тульской области). Датасет можно загрузить в папку /dataset.

🛠️ 3D-модели и конструкция
В архиве Fruit_Picker.zip находятся файлы для 3D-печати (STL/STEP):

Крепление вакуумного захвата — адаптер для ручного пылесоса.

Защитный бампер — из вспененного полиэтилена для защиты дрона о ветки.

Шасси-платформа — из фанеры/пластика для установки сервоприводов.

Все детали спроектированы в Autodesk Fusion 360 и оптимизированы для печати на стандартных 3D-принтерах (пластик PETG, TPU).
